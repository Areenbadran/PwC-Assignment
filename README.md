## Installation:

Installation:
- Open the code with any text editor
- Terminal
- cd Server-side
- npm install
- npm start
- cd Client-side
- cd abc-company
- npm install
- npm start


## How to use?
- Choose the user type if Admin or Customer.
- Sign up to be authorized to all the pages of this website.
- If customer user, Log in to add a complaint.
- Add a complaint by filling the form required information.
- Check the status of this complaint by navigating to "all complaints" button.
- If admin user, Log in to see all the complaints from customers.
- The admin can change the status of a complaint.
- There are three status for the complaints: pending, accepted, and rejected.
- The user can signout and rejoin again.
